Interpolação

A interpolação é representada pelo uso de duplas chaves envolvendo uma variável definida no .ts, uma função, expressão boolean, expressão matemática, entre outos.

Portanto, realiza o one way data binding, que é somente o fluxo do ts para o HTML.

String renderizada com interpolação: {{ url }}

Resultado de expressão matemática (1+1) renderizada com interpolação: {{ 1 + 1 }}

Retorno de uma função renderizada com interpolação: {{ func() }}

Retorno de uma expressão boolean renderizada com interpolação: {{ true && false }}

Property binding

É utilizada em atributos de tags HTML, envolvendo-os com colchetes, faz com que o valor do atributo seja o valor de uma variável que está definida no ts.

Portanto, realiza o one way data binding, que é somente o fluxo do ts para o HTML.
